# Sentinaut

A web application that helps homestay staff analyze guest reviews, classify sentiment, identify feedback themes, and generate suggested management responses using AI.

## Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Backend:** Node.js / Express
* **Database:** MongoDB
* **Authentication:** JWT (JSON Web Tokens)
* **Deployment:** Vercel (Frontend) + Render (Backend)
* **AI API:** Google Gemini Flash / Flash-Lite
